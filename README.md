HerokuCommandRef
================
## へろくコマンドレフ

### 概要
Herokuを勉強している非プログラマー向けのCheatsheetアプリです。
Salesforce1上でよく利用するHerokuコマンドをカテゴリごとに一覧として表示されていますので、通勤時等の勉強のため最適です！
コマンドのデータはSalesforceのデータベースにレコードとして保存されますので、追加・削除・編集等も簡単にできます。
